# Free Deployment

This app can run on Render's free web service tier.

## Render

1. Push this project to a GitHub repository.
2. Go to Render and create a new Blueprint.
3. Connect the repository.
4. Render will detect `render.yaml`.
5. Deploy the `history-hunt-ph` service.

The app uses:

- Build command: `npm install`
- Start command: `npm start`
- Environment variables:
  - `DATA_STORE=file`
  - `JWT_SECRET` generated by Render

## Important Free Hosting Note

The default `DATA_STORE=file` mode stores accounts and scores in `.data/users.json`.
On many free hosts, local files can be reset after redeploys, restarts, or idle sleep.
This is fine for demos, but use a real hosted database for permanent accounts and leaderboard data.
